# Cloud DevOps Dashboard

A real-time monitoring dashboard for Azure DevOps and cloud services.

## Features
- Real-time monitoring of Azure services
- Responsive design
- Auto-refresh capability
- Service status indicators
- Interactive metrics display

## Services Monitored
- **AppGW**: Application Gateway
- **APIM**: API Management  
- **AKS**: Azure Kubernetes Service
- **ADO**: Azure DevOps

## Getting Started

### Prerequisites
- Web browser (Chrome, Firefox, Edge)
- Azure DevOps account (optional)

### Installation
1. Clone this repository
2. Open `index.html` in your browser
3. Or deploy using Azure Static Web Apps
